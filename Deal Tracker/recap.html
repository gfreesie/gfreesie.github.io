Overview

What This Is

An internal team client tracker for mortgage files currently in process. Built as a single self-contained HTML file — no installs, no logins, just open in a browser. Designed for the whole team with a sharp dark UI, fast interactions, and just enough detail to manage files without noise.

Features

What's Built


  • Kanban board — 5 color-coded pipeline columns with stage glow effects and live volume totals
  • Client cards — borrower name, loan amount, loan type, property address, rate lock status, closing date with live countdown
  • Urgency indicators — red pulse dot ≤7 days to close, yellow dot ≤14 days
  • Drag & drop — grab any card and drop into any stage
  • ← → arrows on each card for instant one-step stage movement
  • Detail panel — click any card for full view with pipeline progress bar, advance/move back/delete
  • Add new file — full modal with all fields, rate lock toggle, and stage picker
  • Header stats — live active file count, total pipeline volume, rate lock ratio
  • Search & LO filter — filter board by name, address, or loan officer
  • Dark neural UI — deep dark theme, electric cyan accents, smooth animations
  • Home screen — full-screen constellation canvas, "PIPELINE" title, 3 rotating 3D fractal user icons (GA / SG / RH) to select your session
  • Neural constellation — 90-node interactive canvas background; nodes drift and attract to cursor, edges glow; dims on board screen
  • UV neon borders — pulsing purple-cyan glow on all columns and user cards
  • Smart notes — auto-bullet on click, hacker glitch "Says who?" attribution, timestamped history with show/hide older
